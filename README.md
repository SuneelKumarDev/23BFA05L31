# 23BFA05L31 - Simple Calculator in C

## 📌 Description
This is a **basic calculator program written in C** that performs the following arithmetic operations:  
✔ Addition  
✔ Subtraction  
✔ Multiplication  
✔ Division  

The program runs in a loop until the user chooses to exit. It also handles invalid input cases such as **division by zero** and **invalid choices**.

## 🔧 Features
- Simple **menu-driven** interface.
- Supports **addition, subtraction, multiplication, and division**.
- Handles **division by zero** errors.
- Allows **continuous calculations** until the user exits.
- **User-friendly prompts** for better interaction.

## 🚀 How to Use
### 🔹 Compilation
Use the following command to compile the program:
```sh
gcc MINI-PROJECT-L31.c -o calculator
```
### 🔹 Running the Program
After compilation, run the program using:
```sh
./calculator
```
